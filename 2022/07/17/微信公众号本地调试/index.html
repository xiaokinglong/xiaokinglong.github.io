使用Whistle修改host

# 使用Whistle修改host进行代理

# 安装whistle

npm install -g whistle

# whistle的基础命令

w2 start #启动
w2 stop # 停止
w2 restart # 重启

# 进行配置

在浏览器中输入 http://127.0.0.1:8899/#rules 配置你要代理的域名

在电脑中进行 网络 ---> WiFi ---> 高级 ---> 代理 中进行配置 (如下图) ps: 如果是http的 只需要配置网页代理即可, 端口号为Whistle的端口号

在微信开发者工具中 设置 --> 代理 选择系统代理,然后重启微信开发者工具 出现下面的提示 代表配置成功
