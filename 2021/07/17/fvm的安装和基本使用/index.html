FVM的安装和基本使用

# 步骤1

brew tap leoafarias/fvm
brew install fvm

tips: 可能会出现下面的问题

# 直接在终端中执行下面的命令 等待安装完成即可
sudo rm -rf /Library/Developer/CommandLineTools
sudo xcode-select --install

# 常用的命令

fvm use 2.2.3 # 使用对应的版本的sdk
fvm list # 查看安装所有的版本
fvm install *** # 安装对应的版本的sdk
