# dart 变量的声明

dart同时支持 强类型 和弱类型的变量的声明

弱类型: 申明变量不赋值时 ， 变量可以的给定任意类的值。申明变量的并赋值,后面再次赋值, 值必须是同类型的值，不然报错。 dynamic 除外。

强类型: 在声明变量是的给定值的类型, 后面再次赋值必须是相同类型的值。

一切对象的默认值都是null。

// 声明变量不赋值 可以任意的改变值类型
var name;
a = 'long';
dynamic b;
b = 12;
b = 'long';
Object c;
c = 10;
c = 'long';

// 声明并赋值 
var aciton = 'play';
action = 18; // 直接抛出异常
// Error: A value of type 'int' can't be assigned to a variable of type 'String'.
dynamic action = 'run';
run = 18; // 不报错

// 强类型
int number = 10;

# 常量的声明

常量的声明方式有两种的 使用 final，const 。 跟 JavaScript 一样的常量不能被修改， 并且不能与 var 同时的使用。

// 赋值的区别
final time = new DateTime.now(); 
const time1 = new DateTime.now(); // 报错
// 传递性
final List l1 = [1,2,3];
l1[0] = 9; // 可以修改
const List l2 = [1,2,3];
l2[0] = 9; // 不可以修改
// 在内存中的区别
final List l = [1,2,3];
final List l1 = [1,2,3];
print(identical(l, l1)); // false
// 传递性
const List l = [1,2,3];
const List l1 = [1,2,3];
print(identical(l, l1)); // true
# final 与const 的区别
  1. final 申明的常量是运行时才确定值， 而 const 是在赋值时才确定的。
  2. 使用 final 的申明的常量的子元素是可以修改， 而 const是不能被修改的。如 声明一个 List 的时候 使用final 声明时可以修改的 List 的某一项。
  3. 使用 final声明的常量是在内存中重复创建的， const 是不重复的创建的。****
